The injectivity half of the bijection lemma has a name you should know.

<Callout kind="proposition" title="Cancellation law">
In any group $G$, if $gx = gy$ then $x = y$. (And similarly, if $xg = yg$ then $x = y$.)
</Callout>

This is exactly the statement that $x \mapsto gx$ is injective: equal outputs force equal inputs. The name *cancellation* comes from the fact that you can "cancel" the $g$ from both sides — much like cancelling a common factor in ordinary arithmetic, except that here the justification is that $g$ has an inverse.

<Callout kind="remark" title="Why the name?">
In ordinary arithmetic, $3x = 3y$ implies $x = y$ because we can divide by $3$. In a group, we don't have division — but we have *inverses*, which serve the same purpose. Multiplying both sides of $gx = gy$ by $g^{-1}$ on the left cancels the $g$.
</Callout>
