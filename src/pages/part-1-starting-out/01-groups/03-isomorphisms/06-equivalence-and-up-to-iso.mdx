It's a quick check that $\cong$ is an equivalence relation on groups.

<Callout kind="proposition" title="≅ is an equivalence relation">
For groups $G$, $H$, $K$:

- *(Reflexive)* $G \cong G$ via the identity map.
- *(Symmetric)* If $\phi \colon G \to H$ is an isomorphism, then $\phi^{-1} \colon H \to G$ is also an isomorphism, so $G \cong H$ implies $H \cong G$.
- *(Transitive)* If $\phi \colon G \to H$ and $\psi \colon H \to K$ are isomorphisms, then $\psi \circ \phi \colon G \to K$ is an isomorphism, so $G \cong H$ and $H \cong K$ imply $G \cong K$.
</Callout>

Because we only really care about the *structure* of a group, we usually treat two isomorphic groups as the same object. So phrases like "find all groups of order $n$" really mean "find all groups of order $n$ **up to isomorphism**".
