Just like with groups, looking at *non*-examples sharpens the definition. Each of the following is a subset of $\ZZ = (\ZZ, +)$ that fails exactly one subgroup requirement.

<Callout kind="example" title="Non-examples of subgroups of Z">
- The set $\{0, 1, 2, \dots\}$ of non-negative integers is **not** a subgroup of $\ZZ$, because it does not contain inverses (the additive inverse of $1$ is $-1$, which is missing).
- The set of cubes
$$
\{ n^3 \mid n \in \ZZ \} = \{ \dots, -8, -1, 0, 1, 8, \dots \}
$$
is **not** a subgroup, because it is not closed under addition: $1 + 8 = 9$, and $9$ is not a cube.
- The empty set $\varnothing$ is **not** a subgroup of $\ZZ$, because it lacks an identity element.
</Callout>
