<SlideLayout title="The subgroup generated by an element" part="Part I — Starting Out" chapter="1. Groups · Subgroups">

Here is an especially important example we will return to in later chapters.

<Callout kind="example" title="Subgroup generated by an element">
Let $x$ be an element of a group $G$. Consider the set

$$
\langle x \rangle = \left\{ \dots, x^{-2}, x^{-1}, 1, x, x^2, \dots \right\}.
$$

This is a subgroup of $G$, called the **subgroup generated by** $x$.
</Callout>

It is closed under the group operation (since $x^a \cdot x^b = x^{a+b}$), it contains the identity ($x^0 = 1$), and every element has an inverse inside the set ($(x^a)^{-1} = x^{-a}$). So all three subgroup conditions are met automatically.

Notice the connection to *order*: if $\ord x$ is finite then the powers $\dots, x^{-1}, x^0, x^1, \dots$ start cycling, and the subgroup $\langle x \rangle$ has only finitely many elements.
