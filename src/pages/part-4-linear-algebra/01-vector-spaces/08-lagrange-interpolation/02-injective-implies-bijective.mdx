The Lagrange proof used a slick fact that is worth isolating.

<Callout kind="proposition" title="Equal dimensions: injective ⟺ surjective">
Let $T \colon V \to W$ be a <Term k="linearMap">linear map</Term> with $\dim V = \dim W = n$. The following are equivalent:
- $T$ is injective ($\ker T = \{0\}$).
- $T$ is surjective ($T(V) = W$).
- $T$ is an isomorphism.
</Callout>

This follows directly from rank-nullity: if $\dim \ker T = 0$, then $\dim T(V) = n = \dim W$, so $T(V) = W$.

There is no analogue for sets in general — an injective map $\NN \to \NN$ need not be surjective.
