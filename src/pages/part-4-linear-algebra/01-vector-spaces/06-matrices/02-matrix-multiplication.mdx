Now compose two maps. Given $T$ with matrix $\begin{bmatrix}1&2\\3&4\end{bmatrix}$ and $S$ with matrix $\begin{bmatrix}5&6\\7&8\end{bmatrix}$, what is the matrix of $S \circ T$?

By linearity, we only need $(S \circ T)\!\left(\begin{bmatrix}1\\0\end{bmatrix}\right)$ and $(S \circ T)\!\left(\begin{bmatrix}0\\1\end{bmatrix}\right)$:

$$S\!\left(T\!\left(\begin{bmatrix}1\\0\end{bmatrix}\right)\right) = S\!\left(\begin{bmatrix}1\\3\end{bmatrix}\right) = 1\begin{bmatrix}5\\7\end{bmatrix} + 3\begin{bmatrix}6\\8\end{bmatrix} = \begin{bmatrix}23\\31\end{bmatrix}.$$

Similarly the second column is $\begin{bmatrix}34\\46\end{bmatrix}$. So the matrix of $S \circ T$ is $\begin{bmatrix}23&34\\31&46\end{bmatrix}$.

<Callout kind="remark">
The high-school **recipe** for matrix multiplication — row times column — is exactly this computation. In Napkin, it is a **theorem** derived from function composition, not an arbitrary definition.
</Callout>

Since function composition is associative, matrix multiplication is associative. Since the identity function is encoded as $\begin{bmatrix}1&0\\0&1\end{bmatrix}$, that matrix deserves its name.
