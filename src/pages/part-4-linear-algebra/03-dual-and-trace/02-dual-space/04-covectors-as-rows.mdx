For $V = \mathbb{R}^n$, there is a clean dictionary between elements of <Term k="dualSpace">$V^\vee$</Term> and row vectors.

| Object | Representation |
|--------|---------------|
| Vector $v \in V = \mathbb{R}^n$ | Column matrix $n \times 1$ |
| Covector $\xi \in V^\vee$ | Row matrix $1 \times n$ |
| Evaluation $\xi(v)$ | Row × column = scalar |

Any $\xi = a_1 e_1^\vee + \cdots + a_n e_n^\vee$ acts by $\xi(v) = a_1 v_1 + \cdots + a_n v_n$, which is precisely the dot product $[a_1 \cdots a_n] \cdot [v_1 \cdots v_n]^T$.
