<SlideLayout title="Hom(V,k) is V∨">

The isomorphism $\operatorname{Hom}(V,W) \cong V^\vee \otimes W$ specialises nicely when $W = k$.

$$\operatorname{Hom}(V, k) \cong V^\vee \otimes k \cong V^\vee$$

This is exactly the definition of the <Term k="dualSpace">dual space</Term>: $V^\vee = \operatorname{Hom}(V, k)$. So the notation $V^\vee$ and the notation $\operatorname{Hom}(V,k)$ are the same object.

Plugging in $V = W$ also gives a particularly important case for the trace:

$$\operatorname{Hom}(V, V) \cong V^\vee \otimes V$$

Every linear map $T: V \to V$ is represented as an element of <Term k="dualSpace">$V^\vee$</Term> $\otimes V$, and this identification is used to define the trace.
