<SlideLayout title="From V∨ ⊗ W to linear maps">

Here is how to interpret an element of <Term k="dualSpace">$V^\vee$</Term> $\otimes$ $W$ as a linear map $V \to W$: feed the vector into the $V^\vee$ factor.

**Example:** Suppose $V$ has basis $e_1, e_2, e_3$ and $W$ has basis $f_1, \dots, f_5$. Take the element

$$e_1^\vee \otimes (f_2 + 2f_4) + 4e_2^\vee \otimes f_5 \;\in\; V^\vee \otimes W.$$

This represents the map $v \mapsto e_1^\vee(v)\cdot(f_2 + 2f_4) + 4e_2^\vee(v)\cdot f_5$.

In general, any element $\xi_1 \otimes w_1 + \cdots + \xi_m \otimes w_m$ represents:

$$v \;\mapsto\; \xi_1(v)\,w_1 + \cdots + \xi_m(v)\,w_m$$

<Callout kind="theorem" title="V∨ ⊗ W ≅ Hom(V,W)">
Let $V$ and $W$ be finite-dimensional vector spaces. The map
$$\Psi \colon V^\vee \otimes W \to \operatorname{Hom}(V,W)$$
defined above is an isomorphism of vector spaces.
</Callout>
