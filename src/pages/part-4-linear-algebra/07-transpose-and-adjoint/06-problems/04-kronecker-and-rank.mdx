  prompt="Let M be an m×n matrix of complex numbers. Show that M†M and M have equal rank. Try to find a basis-free proof by showing ker(T†T) = ker(T) for any linear map T."
  hint="Show directly that if T†T(v) = 0 then T(v) = 0, using the inner product."
  solution="It suffices to show ker(T†T) = ker(T), since then rank(T†T) = dim V − dim ker(T†T) = dim V − dim ker(T) = rank(T). The inclusion ker(T) ⊆ ker(T†T) is clear (if T(v) = 0 then T†T(v) = T†(0) = 0). For the other direction: suppose T†T(v) = 0. Then ⟨v, T†T(v)⟩ = 0, i.e. ⟨T(v), T(v)⟩ = 0. By positive-definiteness of the inner product, T(v) = 0. So ker(T†T) ⊆ ker(T). Hence ker(T†T) = ker(T), and rank(M†M) = rank(M)."
