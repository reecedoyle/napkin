<Problem
  client:load
  id="trn-prob-complex-conj-space"
  prompt="Let V = (V, +, ·) be a complex vector space. Define the complex conjugate vector space V̄ = (V, +, ∗) by c ∗ v = c̄ · v. Show that for any sesquilinear inner form on V, the map V̄ → V∨ given by v ↦ ⟨−, v⟩ is an isomorphism of complex vector spaces."
  hint="Check that v ↦ ⟨−, v⟩ is linear as a map from V̄: compute what happens when you scale by c in V̄ (using the conjugated scalar multiplication ∗)."
  solution="The map φ: V̄ → V∨ is given by φ(v) = ⟨−, v⟩. We check linearity over ℂ in V̄: φ(c ∗ v) = ⟨−, c̄v⟩ = c⟨−, v⟩ = c · φ(v), using sesquilinearity. So φ is linear from V̄ to V∨. By the same argument as for the real case, φ is injective (take v as the input: ⟨v, v⟩ = 0 only if v = 0) and dim V̄ = dim V∨, so φ is an isomorphism."
  difficulty="standard"
/>
