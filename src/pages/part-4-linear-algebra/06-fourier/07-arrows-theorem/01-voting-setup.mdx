<SlideLayout title="Arrow's theorem — the voting setup">

Binary Fourier analysis has a surprising application in social choice theory.

**Setup:** $n$ voters each rank three candidates $A$, $B$, $C$. Each voter $i$ specifies a tuple $(x_i, y_i, z_i) \in \{\pm 1\}^3$ where:
- $x_i = 1$ if voter $i$ prefers $A$ over $B$ (else $-1$),
- $y_i = 1$ if voter $i$ prefers $B$ over $C$ (else $-1$),
- $z_i = 1$ if voter $i$ prefers $C$ over $A$ (else $-1$).

We only allow the $3! = 6$ *consistent* (transitive) rankings: the paradoxical votes $(1,1,1)$ and $(-1,-1,-1)$ (which would represent a preference cycle) are excluded.

<Callout kind="remark" title="What is a 'paradoxical' outcome?">
A global outcome $(f(x_\bullet), g(y_\bullet), h(z_\bullet)) \in \{\pm1\}^3$ is paradoxical if all three coordinates agree — $A \succ B$, $B \succ C$, and $C \succ A$ simultaneously. This is the "Condorcet cycle" that voting systems try to avoid.
</Callout>

A <Term k="votingMechanism">voting mechanism</Term> is three functions $f, g, h \colon \{\pm1\}^n \to \{\pm1\}$ aggregating the voters' pairwise preferences into a global preference.

</SlideLayout>
