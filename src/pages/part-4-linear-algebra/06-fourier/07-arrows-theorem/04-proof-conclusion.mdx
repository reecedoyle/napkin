<SlideLayout title="Arrow's theorem — conclusion of proof">

All inequalities in the chain on the previous slide must be **sharp** (equalities). This forces:

1. The factor $\left(\frac{1}{3}\right)^{|S|}$ must not drop below $\frac{1}{3}$, so $|S| \le 1$ for all contributing terms — meaning $\wh{f}$, $\wh{g}$, $\wh{h}$ are supported on sets $S$ of size at most $1$.

2. Since $f, g, h \colon \{\pm1\}^n \to \{\pm1\}$ take values $\pm 1$, a function supported only on singletons must be a single monomial $\pm x_k$ for some $k$.

3. Since $(f, g, h)$ always yields a consistent (non-paradoxical) outcome, the same dictator $k$ and the same sign must appear in all three.

<Callout kind="remark" title="Moral">
Fourier analysis converts a combinatorial impossibility statement into a string of inequalities, each of which is a standard algebraic fact. The equality conditions pin down the structure of $f$, $g$, $h$ exactly.
</Callout>
