<Callout kind="theorem" title="Bolzano-Weierstraß">
A subset $S \subseteq \mathbb{R}^n$ is <Term k="compact">compact</Term> if and only if it is **closed and bounded**.
</Callout>

**Why closed and bounded $\Rightarrow$ compact:** Since $S$ is bounded, it fits inside a box $[a_1, b_1] \times \cdots \times [a_n, b_n]$. Each interval $[a_i, b_i]$ is compact; by Tychonoff their product is compact. Since $S$ is a closed subset of this compact box, $S$ is compact.

**Warning:** This only works in $\mathbb{R}^n$! In other metric spaces, closed and bounded can fail to imply compact.

<Callout kind="example" title="Closed and bounded but not compact">
Take any infinite set $S = \{s_1, s_2, \dots\}$ with the <Term k="discreteMetric">discrete metric</Term>. Then $S$ is closed and bounded (all points are distance 1 apart), but the sequence $s_1, s_2, \dots$ has no convergent subsequence — it is not compact.
</Callout>
