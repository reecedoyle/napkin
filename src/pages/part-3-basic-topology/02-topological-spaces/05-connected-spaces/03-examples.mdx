<SlideLayout title="Examples of connected and disconnected spaces">

<Callout kind="example" title="Disconnected spaces">
- $[0,1] \cup [2,3]$ is disconnected: $[0,1]$ is clopen in it (its complement $[2,3]$ is also clopen).
- Any discrete space on more than one point is disconnected: every subset is clopen.
- $\mathbb{Q}$ is disconnected: the set $\{x \in \mathbb{Q} : x^2 < 2\}$ is clopen in $\mathbb{Q}$ (its complement is $\{x \in \mathbb{Q} : x^2 > 2\}$, also open). The "gap" at $\sqrt{2}$ splits $\mathbb{Q}$.
</Callout>

<Callout kind="example" title="Connected spaces">
The interval $[0,1]$ is connected. So is $\RR$, $\RR^n$, and any convex subset of $\RR^n$. (Proofs of these require a bit more work, typically using the intermediate value theorem.)
</Callout>
