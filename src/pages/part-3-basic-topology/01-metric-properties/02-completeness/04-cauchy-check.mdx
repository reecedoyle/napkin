<SlideLayout title="Identifying Cauchy sequences">

Let's make the Cauchy condition concrete. Working in <Term k="RR">$\mathbb{R}$</Term> with the usual metric $d(x,y) = |x - y|$:

<Callout kind="example">

- $a_n = \tfrac{1}{n}$: Is this Cauchy? For large $m, n$ we have $|a_m - a_n| = |\tfrac{1}{m} - \tfrac{1}{n}|$, which can be made as small as we like. Yes, Cauchy — and it converges to $0 \in \mathbb{R}$.
- $b_n = (-1)^n$: alternates between $-1$ and $1$. The distance $|b_m - b_n|$ equals $2$ infinitely often, so **not** Cauchy.
- The partial sums $s_n = \sum_{k=1}^n \tfrac{1}{k^2}$: Cauchy in $\mathbb{R}$, converging to $\pi^2/6$.

</Callout>
