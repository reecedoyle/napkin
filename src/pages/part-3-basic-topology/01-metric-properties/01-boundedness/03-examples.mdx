<SlideLayout title="Examples of bounded and unbounded spaces">

<Callout kind="example" title="Bounded spaces">

- Finite intervals like $[0,1]$ and $(a,b)$ are bounded.
- The unit square $[0,1]^2$ is bounded.
- <Term k="discreteMetric">A discrete space</Term> on **any** set is bounded: all distances are either 0 or 1, so $D = 1$ works.

</Callout>

<Callout kind="example" title="Unbounded spaces">

- <Term k="RRn">$\mathbb{R}^n$</Term> is not bounded for any $n \ge 1$.
- <Term k="NN">$\mathbb{N}$</Term> (with the usual metric) is not bounded, even though it is homeomorphic to the discrete space on an infinite set!

</Callout>

The last point is a warning: boundedness is not a topological property — it is not preserved by homeomorphism.
