<SlideLayout title="Why we need an integral domain">

The field-of-fractions construction requires $R$ to be an integral domain. Let's see why zero divisors cause trouble.

In the construction, we need $bd \neq 0$ when $b, d \neq 0$ (so the denominator of $a/b + c/d$ is valid). This is exactly the "no zero divisors" property.

We also need the equivalence relation $a/b \sim c/d \iff ad = bc$ to be transitive: if $a/b \sim c/d$ and $c/d \sim e/f$, we need $a/b \sim e/f$. Working through this requires cancellation — which holds in integral domains but fails with zero divisors.

<Callout kind="remark">
If $R$ has zero divisors, you can try to build a "localisation" more carefully, but the naive fraction construction breaks. Integral domains are exactly the rings where fractions work the way you expect.
</Callout>
