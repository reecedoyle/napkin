<SlideLayout title="Maximal ideals and quotient fields">

Maximal ideals have the same clean characterisation as prime ideals, one level up the hierarchy.

<Callout kind="theorem" title="Maximal ideal ⟺ quotient is a field">
An ideal $I \subsetneq R$ is maximal if and only if the quotient ring $R/I$ is a field.
</Callout>

The logic: a field has only two ideals, $(0)$ and itself. So asking "$I$ is maximal in $R$" is equivalent to asking "there are no proper ideals between $I$ and $R$" — which translates to "the only ideals of $R/I$ are $\bar 0$ and $R/I$" — which means $R/I$ is a field.

**Fast examples:**
- $\mathbb{Z}[x]/(x, 5) \cong \mathbb{F}_5$ (mod out by $x$ to kill the polynomial, mod out by $5$ to get the field). So $(x, 5)$ is maximal. ✓
- $\mathbb{Z}[x]/(x) \cong \mathbb{Z}$ (integral domain, not a field). So $(x)$ is prime but **not** maximal. ✓
- $\mathbb{C}[x]/(x) \cong \mathbb{C}$ (a field). So $(x)$ is maximal in $\mathbb{C}[x]$. ✓
