<SlideLayout title="When is (0) maximal?">

As with prime ideals, we can ask: when is $(0)$ maximal?

If $(0)$ is maximal, then $R/(0) \cong R$ is a field. Conversely, if $R$ is a field then $(0)$ is the only maximal ideal — it is already maximal. So:

<Callout kind="proposition">
The zero ideal $(0)$ is maximal in $R$ if and only if $R$ is a field.
</Callout>

Together with the earlier result about prime ideals, we get a neat summary:
- $(0)$ is prime $\iff$ $R$ is an integral domain.
- $(0)$ is maximal $\iff$ $R$ is a field.
