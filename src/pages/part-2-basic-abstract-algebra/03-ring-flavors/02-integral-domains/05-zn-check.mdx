<SlideLayout title="Which ℤ/nℤ are integral domains?" part="Part II — Basic Abstract Algebra" chapter="5. Flavors of rings · Integral domains">

We claimed that <Term k="Zc">$\mathbb{Z}/n\mathbb{Z}$</Term> is an integral domain (equivalently, a field) exactly when $n$ is prime. Let's think through why.

<Callout kind="proposition">
$\mathbb{Z}/n\mathbb{Z}$ is an integral domain if and only if $n$ is prime.
</Callout>

**If $n$ is prime:** every nonzero $\bar a$ satisfies $\gcd(a, n) = 1$, so Bézout gives an inverse. Hence it's a field, hence an integral domain.

**If $n$ is composite:** write $n = ab$ with $1 < a, b < n$. Then $\bar a \neq 0$ and $\bar b \neq 0$ in $\mathbb{Z}/n\mathbb{Z}$, but $\bar a \cdot \bar b = \overline{ab} = \bar n = \bar 0$. So $\bar a$ is a zero divisor.
