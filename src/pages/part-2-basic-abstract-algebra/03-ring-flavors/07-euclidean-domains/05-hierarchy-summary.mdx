<SlideLayout title="The hierarchy — summary">

We now have the full hierarchy:

$$\text{Fields} \subsetneq \text{Euclidean domains} \subsetneq \text{PIDs} \subsetneq \text{UFDs} \subsetneq \text{Integral domains}$$

Concrete witnesses that each inclusion is strict:

| Ring | Euclidean? | PID? | UFD? | Integral domain? |
|---|---|---|---|---|
| $\mathbb{Q}$ | Yes (it's a field) | Yes | Yes | Yes |
| $\mathbb{Z}[i]$ | Yes | Yes | Yes | Yes |
| $\mathbb{Z}\!\left[\tfrac{1+\sqrt{-19}}{2}\right]$ | **No** | Yes | Yes | Yes |
| $\mathbb{Z}[x]$ | No | **No** ($(2,x)$ not principal) | Yes | Yes |
| $\mathbb{Z}[\sqrt{-5}]$ | No | No | **No** (6 has two factorisations) | Yes |
