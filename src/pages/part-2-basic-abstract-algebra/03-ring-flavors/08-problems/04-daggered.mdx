Two harder problems: the first uses a clever finiteness argument to show all nonzero prime ideals of a quadratic integer ring are maximal; the second classifies ideals of a specific localisation.

<Problem
  client:load
  id="flv-prob-dedekind-sample"
  prompt="How many prime ideals of ℤ[√2017] are NOT maximal?"
  hint="Show that ℤ[√2017]/I has finitely many elements for any nonzero prime ideal I. A finite integral domain is a field, so I is maximal."
  solution="Exactly one: the zero ideal (0), which is prime (ℤ[√2017] is an integral domain) but not maximal (ℤ[√2017] is not a field).

Every other prime ideal is maximal: let I be a nonzero prime ideal containing some nonzero element a + b√2017. Then I also contains the norm N = a² - 2017b² (since (a + b√2017)(a - b√2017) = a² - 2017b²), which is a nonzero integer. So in R/I we work modulo the integer n = |a² - 2017b²| ≠ 0. Every element of R/I has the form x + y√2017 with x, y ∈ {0, 1, ..., n-1}, giving finitely many elements. Since I is prime, R/I is an integral domain. A finite integral domain is a field (problem above). So R/I is a field, meaning I is maximal."
  difficulty="daggered"
/>

<Problem
  client:load
  id="flv-prob-localisation-ideals"
  prompt="Let R be the set of rational numbers q such that when q is written in lowest terms, the denominator is not a multiple of 5. Then R is a ring under ordinary + and ×. Classify all ideals of R. Which are prime? Which are maximal?"
  solution="The ideals of R are: (0), R = (1), and (5^n) = 5^n R for each n ≥ 1. (Every ideal is generated by the appropriate power of 5.)

(0) is prime (R is an integral domain — it sits inside ℚ, which has no zero divisors).
(5) is maximal: R/(5) ≅ 𝔽_5 (a field), since modding out by 5 kills all multiples of 5, and what remains is {0,1,2,3,4} mod 5.
For n ≥ 2, (5^n) is neither prime nor maximal: 5 · 5^(n-1) = 5^n ∈ (5^n) but 5 ∉ (5^n) and 5^(n-1) ∉ (5^n)."
  difficulty="standard"
/>

<Problem
  client:load
  id="flv-prob-artinian"
  prompt="Recall that a ring R is Artinian if there is no infinite strictly descending chain of ideals I_1 ⊋ I_2 ⊋ I_3 ⊋ ···.
(a) Show that if R is Artinian and an integral domain, then R is a field.
(b) More generally, show that every prime ideal in an Artinian ring is maximal."
  solution="(a) Let x ∈ R, x ≠ 0. The ideals (x) ⊇ (x²) ⊇ (x³) ⊇ ··· form a descending chain. By Artinian, it stabilises: (x^n) = (x^(n+1)) for some n. So x^n = r · x^(n+1) for some r ∈ R, i.e. x^n(1 - rx) = 0. Since R is an integral domain and x ≠ 0, we get x^(n-1)(1 - rx) = 0 by cancellation (inducting), eventually 1 - rx = 0, so rx = 1. Thus x is a unit, and R is a field.

(b) Let P be a prime ideal of an Artinian ring R. Then R/P is an Artinian integral domain (Artinian passes to quotients, and prime gives integral domain). By part (a), R/P is a field. Hence P is maximal."
  difficulty="standard"
/>
