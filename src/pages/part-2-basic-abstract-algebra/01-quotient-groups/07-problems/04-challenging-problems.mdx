<Problem
  client:load
  id="quot-prob-primes-presentation"
  prompt="[PRIMES 2018] Let G be the group with presentation G = ⟨a, b, c | ab = c²a⁴, bc = ca⁶, ac = ca⁸, c²⁰¹⁸ = b²⁰¹⁹⟩. Determine the order of G."
  hint="Try to show G is the dihedral group of order 18. There is not much group theory content here — mostly algebraic manipulation with the relations."
  solution="|G| = 18.

Key steps:
1. By induction from ac = ca⁸, get aⁿc = ca^{8n} for all n ≥ 1.
2. Use associativity (ab)c = a(bc): substituting ab = c²a⁴ and bc = ca⁶ gives c²a⁴·c = c²·a^{32}, so a^{14} = c·a^4·c = c²a^{32}.
   Hence c² = a^{−18}.
3. From ab = c²a⁴ and c² = a^{−18}: b = a^{−18}a^{−1}a⁴ = a^{−15}.
4. From c^{2018} = b^{2019} = a^{−15·2019}: combined with c^2 = a^{−18} gives a^{1009·18} = a^{2019·15}, leading to a^{12123} = 1.
5. From bc = ca⁶: a^{−15}c = ca⁶, so a^{−15}c² = c·a⁶c = c²·a^{48}, giving a^{−33} = a^{30}, so a^{63} = 1.
6. Since gcd(12123, 63) = 9, we get a⁹ = 1, and then c² = a^{−18} = 1.
7. The presentation simplifies to ⟨a, c | a⁹ = c² = 1, ac = ca^{−1}⟩ = D_{18}, the dihedral group of order 18."
  difficulty="starred"
/>
