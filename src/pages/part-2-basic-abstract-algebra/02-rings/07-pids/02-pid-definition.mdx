<Callout kind="definition" title="Principal ideal domain (PID)">
An integral domain in which every ideal is principal (generated by a single element) is called a <Term k="pid">**principal ideal domain**</Term>, abbreviated **PID**.
</Callout>

"Integral domain" will be defined precisely in Chapter 5; for now think "commutative ring with no zero divisors". The examples below are all PIDs.

<Callout kind="example" title="PIDs and non-PIDs">
- $\mathbb{Z}$ is a PID: every ideal is $n\mathbb{Z} = (n)$.
- For any field $k$, the polynomial ring $k[x]$ is a PID. ($\mathbb{Q}[x]$, $\mathbb{R}[x]$, $\mathbb{C}[x]$ are all PIDs.)
- $\mathbb{Z}[x]$ is **not** a PID: $(x, 2015)$ is not principal.
- $\mathbb{C}[x, y]$ is **not** a PID: $(x, y)$ is not principal.
</Callout>

PIDs matter because, in a PID, one can always take the "greatest common factor" of any two elements — mirroring $\gcd$ in $\mathbb{Z}$. This is proved via Bézout-style arguments; Chapter 5 develops this systematically.
