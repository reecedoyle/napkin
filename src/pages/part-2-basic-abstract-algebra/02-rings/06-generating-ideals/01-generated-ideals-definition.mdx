<Callout kind="definition" title="Ideal generated by elements">
Let $R$ be a ring and $x_1, \dots, x_n \in R$. The **ideal generated** by $x_1, \dots, x_n$ is

$$\Term{generatedIdeal}{(x_1, x_2, \dots, x_n)} = \{\, r_1 x_1 + r_2 x_2 + \cdots + r_n x_n \mid r_i \in R \,\}.$$

This is the smallest ideal of $R$ that contains all the $x_i$.
</Callout>

When $I = (x)$ has a single generator, the ideal consists exactly of the **multiples** of $x$: elements of the form $rx$ for $r \in R$. We call this a **principal ideal**.

<Callout kind="proposition" title="Proper ideal ⟺ no units">
An ideal $I \subseteq R$ is proper ($I \neq R$) if and only if it contains no unit of $R$.
</Callout>

Proof: if $u \in I$ is a unit with $u^{-1} \in R$, then $1_R = u^{-1} \cdot u \in I$ (by absorption), so $I = R$. Conversely, if $I$ contains no unit it cannot equal $R$ (which contains $1_R$).

</SlideLayout>
